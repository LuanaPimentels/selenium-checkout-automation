# Selenium Checkout Automation

Automated end-to-end test for an e-commerce checkout flow using Python and Selenium.

This project simulates a user completing a purchase on the SauceDemo test website.

## 🚀 Technologies Used

- Python
- Selenium WebDriver
- ChromeDriver

## 📋 Test Scenario

The automated script performs the following steps:

1. Opens the browser in incognito mode
2. Accesses the website https://www.saucedemo.com
3. Logs in with a test user
4. Selects a product
5. Adds the product to the cart
6. Opens the cart
7. Proceeds to checkout
8. Fills in customer information
9. Completes the purchase
10. Saves a screenshot of the result

## ▶️ How to Run

1. Install Python

2. Install Selenium
pip install selenium

3. Download ChromeDriver compatible with your Chrome version

4. Run the script
python teste_checkout.py
